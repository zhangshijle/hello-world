## 项目简介： 

```
1.将基于go语言开发的项目进行代码编译、构建成容器镜像并上传到京东云镜像仓库、部署到京东云kubernetes环境, 并实现基于ALB在公网访问

2.在京东云云主机部署jenkins，用于实现kubernetes环境中的APP代码部署和回滚

3.编写脚本、基于jenkins实现go语言代码部署和回滚
```
